## üß™ **Technology Behind Shopify**

Shopify‚Äôs technology is designed for speed, scalability, and developer flexibility.

**Core Technologies:**

* Originally built with **Ruby on Rails**
* Uses **MySQL** with advanced sharding and scaling architecture
* Frontend themes use **Liquid**
* Modern headless solutions use **Hydrogen (React + Remix)** and **Storefront API**
* Hosted on a global CDN for fast performance

**Key Objectives:**

* Make e-commerce fast
* Allow customization without sacrificing performance
* Support millions of merchants without downtime
