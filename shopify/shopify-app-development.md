# Shopify App Development

## Shopify App Development

## **Shopify CLI for apps**

Shopify CLI is a command-line interface tool that helps you build Shopify apps. It quickly generates Shopify apps and generates app extensions. You can also use it to automate many common development tasks.

[About Shopify CLI for apps](https://shopify.dev/docs/apps/build/cli-for-apps)

## Getting started

[Scaffold an app](https://shopify.dev/docs/apps/build/scaffold-app)

```bash
# Install Shopify CLI
npm install -g @shopify/cli@latest

# Check installed Shopify version
shopify version # 3.63.2

# Get help
shopify help

# Initialize shopify app
shopify app init

# Run dev server
shopify app dev
```

üí° You can specify the \[app template]\(https://shopify.dev/docs/api#app-templates) that you want to use with the \`--template\` flag. You can specify a Shopify app template, or pass a URL to a template hosted in GitHub.

```bash
# Start with Shopify app node template
shopify app init --template=node

# Checkout GitHub repo for shopify app template node
# https://github.com/Shopify/shopify-app-template-node
```
